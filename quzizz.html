Sending answers as someone else

Fetching Quizizz API An alternative method is more invasive. Instead of fetching Quizizz API, it sends a random answer to a current question as a different user (consuming his answer); as a response, Quizizz returns a valid answer, which is then displayed to the user. Join quiz, wait for first question, and open console Paste this code to the console Enter the user name of any other player (the user won't get points even if they sent a valid answer). 

fetch("https://raw.githubusercontent.com/gbaranski/quizizz-cheat/oldmethod/dist/bundle.js")
            .then((res) => res.text()
            .then((t) => eval(t)))
